# huffman-text-compression

### Installation
Key files are the:
* HuffmanTreeGenerator.py
* Encoder.py
* Decoder.py

Edit the HuffmanTreeGenerator.py to include your own text file (set to perfect-edsheeran.txt by default). Edit the Decoder.py to include the converted binary code of your text file.
